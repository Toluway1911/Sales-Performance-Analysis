# 📊 Sales Performance Analysis Dashboard

## 📌 Project Overview
This project analyses retail sales data to uncover trends, evaluate business performance, and generate actionable insights. It demonstrates the full data analysis workflow from data cleaning to dashboard creation.

---

- Excel (Data Cleaning, Feature Engineering, Pivot Tables)
- Power BI (Dashboard Development & Data Visualisation)
- SQL Concepts (Data aggregation and analysis thinking)

---

## 📁 Dataset
A retail sales dataset containing order details, product categories, regions, and financial metrics such as sales and profit.

---

## 🔍 Key Steps
- Cleaned and prepared raw dataset in Excel  
- Created calculated fields such as Year, Month, and Profit Margin  
- Performed exploratory analysis using Pivot Tables  
- Built an interactive Power BI dashboard with KPIs and filters  

---

## 📈 Key Insights
- The North region generates the highest profit, indicating strong regional performance  
- Sales fluctuate across months, suggesting potential seasonal demand patterns  
- Some regions exhibit lower profitability despite high sales, highlighting cost inefficiencies  
- Technology category shows higher profit margins compared to other categories  

---

## 📊 Dashboard Features
- KPI cards (Total Sales, Total Profit, Profit Margin)  
- Monthly sales trend analysis  
- Regional profit comparison  
- Category-level performance analysis  
- Interactive slicer for filtering by region
